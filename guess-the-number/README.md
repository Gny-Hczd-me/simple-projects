# Guess the Number

A simple Python game where the computer randomly selects a number within a user-defined range, and you try to guess it within a limited number of attempts.

## How to Play

1. Run the script:
python `guess-the-number.py`
2. The game will ask if you want to play. Enter yes to start.
3. Enter the starting and ending numbers of the interval.
4. Enter the number of attempts you want.
5. Try to guess the randomly chosen number. The game will tell you if your guess is wrong and how many attempts are left.
6. If you guess the number correctly, you win!
